# 🏥 Physio Rehab Center (PRC)
### **Sistem Informasi Manajemen Klinik Fisioterapi**

---

## 📝 Deskripsi Project
**Physio Rehab Center (PRC)** adalah aplikasi manajemen operasional klinik fisioterapi yang dibangun menggunakan **Java SE (Swing GUI)** dan didukung oleh sistem basis data **MySQL/MariaDB**. 

Sistem ini dirancang khusus untuk mengotomatisasi siklus pelayanan klinis secara *end to end* dengan menerapkan arsitektur **Pay per Visit** guna menjamin konsistensi pencatatan keuangan, presisi resep kefarmasian (Manajemen Obat), serta pelacakan perkembangan klinis pasien (*Clinical Records*) secara *real-time*. Seluruh arsitektur kode program dibangun di atas pilar utama *Object-Oriented Programming* (OOP) meliputi *Inheritance, Polymorphism,* dan *Encapsulation*.
